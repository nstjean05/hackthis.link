Welcome to my testsite!

What is this site for?

This (under construction) site will be used to help semi-technical people learn some of the ins-and-outs of cybersecurity and OSINT. On the right, you'll see a dropdown menu, containing several levels. Each of these focuses on a different aspect of cybersecurity, and generally gets more involved as you progress.

How can I play?

Each of the dropdowns has a short description of the problem, and a box for text entry. In here, you will enter what is called a "flag". These "flags" are found throughout this site in the form FLAG{flag_code}. Pasting the flag into the box will result in the completion of each level!

1. A Polar Bear in a Snowstorm

One of the things we use in cybersecurity is the terminal, which allows us to directly interact with websites, rather than using their GUI. While maybe not as pretty, the terminal gives us many tools to see what a website or server is doing or made of. Commands like curl or wget can be pointed at a website's url to return its html.
